# RoboScene+

3D Gaussian Splatting reconstruction of a bedroom from a phone video, with semantic object labeling.

👉 **[Live Demo](https://huggingface.co/spaces/jrameshs/roboscene-plus)**

Built with [gsplat.js](https://github.com/huggingface/gsplat.js) · 3.74M Gaussians · 10 semantic classes
